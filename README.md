# Academic Website - AI for Weather & Climate Research

Personal academic website for Minglang Zhang, showcasing research in AI applications for meteorology and climate science.

## Features

- Clean, responsive design using Minimal Mistakes theme
- Publications listing with categorization
- Research projects showcase
- About page with biography and CV information
- Contact information
- Optimized for academic content

## Local Development

1. Install Ruby and Bundler
2. Install dependencies:
   ```bash
   bundle install
   ```
3. Run the site locally:
   ```bash
   bundle exec jekyll serve
   ```
4. Visit `http://localhost:4000`

## Deployment

This site is configured for GitHub Pages. Simply push to the `main` branch of your `zhangminglang42.github.io` repository.

## Customization

- Edit `_config.yml` for site settings
- Update `_pages/` for content pages
- Modify `_data/navigation.yml` for menu structure
- Add publications to `_pages/publications.md`
- Add projects to `_pages/projects.md`
